# E-CommerceApp

# Those used in the project

* MVVM with Clean Architecture
* Coroutines
* Navigation Components
* Dependency Injection - Dagger-Hilt
* Room Database
* Firebase(Firestore/FirebseAuth)
* ViewPager
* Glide
* Data Binding
* View Binding
* Lottie
